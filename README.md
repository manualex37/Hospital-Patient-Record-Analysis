# Hospital-Patient-Record-Analysis
This project analyzes a real-world dirty hospital patient dataset to identify  meaningful healthcare insights through data cleaning, feature engineering,  exploratory data analysis, and visualization. 

Hospitals generate large volumes of patient data, but dirty or incomplete 
records reduce analytical value. This project focuses on cleaning and 
analyzing patient records to support healthcare insights.
